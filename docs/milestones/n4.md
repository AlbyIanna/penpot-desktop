## What was built

### N4a — palette + Peek + grid fix
- **Grid diff/patch fix (N3 debt)** — `apps/desktop/src/home.html`. The 5 s
  refresh and the filter no longer `innerHTML=""` the grid. A keyed card cache
  (`boardId → {el, data}`) reconciles in place: stale cards removed, new ones
  inserted, changed ones patched (name/project/deep-link/thumb only when the
  value actually moved), order fixed with `insertBefore`. Scroll position is
  never touched and the focused card is re-`focus({preventScroll:true})`ed after
  the patch. This is the substrate Peek's arrow-key flipping needs.
- **Palette ranking** — `crates/vault-index/src/palette.rs`. A pure fzf-style
  fuzzy matcher over the vault corpus (projects / files / boards assembled from
  the index rows + manifest). Subsequence-required scoring with word-start,
  label-start and contiguity bonuses; a board outranks an equally-named file or
  project so Enter lands you on a canvas. Exposed at
  `GET /__api/vault/palette?q=&limit=` (`crates/vault-index/src/http.rs`), each
  hit carrying `kind` + the **exact** deep link (the Enter payload).
- **Palette page** — `apps/desktop/src/palette.html`, served at `/__palette`.
  Plain HTML/JS; as-you-type ranking, arrow/Enter selection, and the verbs:
  Open, Present (viewer), Reveal in Finder, Export board (opens the `.exports`
  render), Copy board link (with the boot-fresh team-id staleness caveat),
  Checkpoint now, and the deferring stubs New-from-template (→ N6) and Switch
  vault (→ N5) — both do nothing destructive.
- **Peek** — in `home.html`. Space on a focused card opens a full-size preview
  served from the board's `.exports` render (`/__api/vault/thumb`); ← → flip to
  sibling boards, Esc closes, "Present" opens
  `/#/view?file-id&page-id&frame-id&section=interactions` in the main window.
- **Global shortcut + overlay window** — `apps/desktop/src/overlay.rs` +
  `main.rs`. `tauri-plugin-global-shortcut` binds a configurable default
  (Cmd+K on macOS, Ctrl+K elsewhere; override with
  `PENPOT_LOCAL_PALETTE_SHORTCUT`, e.g. `ctrl+shift+p`) that toggles an
  always-on-top palette overlay window pointing at `/__palette`. The palette is
  ALSO reachable from the tray ("Quick open…") so it is usable/testable without
  the shortcut (PLAN2.md risk 7).

### N4b — Checkpoint now
- `apps/desktop/src/checkpoint.rs` + `POST /__api/vault/checkpoint`, plus a tray
  "Checkpoint now" entry. Pure decision table (`decide`) over a probed
  `RepoState`:
  - **No repo** → init via the shipped `designs-git-init.sh` machinery
    (`gitinit.rs`) + exactly one commit (manifest + `.penpot` dirs).
  - **Clean + changes** → one labeled commit (`Checkpoint <RFC3339>`),
    only ever *adds* a commit (no amend/rebase/reset).
  - **Clean + no changes** → clean no-op.
  - **In-progress/dirty** (rebase, merge, cherry-pick, revert, bisect, detached
    HEAD, unmerged conflict paths) → **loud refusal** (HTTP 409), touching
    nothing.

## What is gated where

- **Headlessly gateable (local + CI):** palette ranking/verbs over HTTP; the
  palette-Enter navigation; the grid-scroll/focus-preservation assert; Peek
  preview bytes + hash; reveal; the checkpoint git decision table (unit) and its
  live-endpoint proof (gate `(g)`); the viewer route (routes-gate).
- **Manual-QA only (NOT a local exit criterion):** the global-shortcut firing
  and the overlay window not stealing focus mid-edit. `tauri-driver` has no
  macOS support, so no local criterion may depend on it; Linux CI is where this
  automation would live (honest asymmetry, as in M4).

### Manual-QA checklist (OS integration)
1. Launch the app; wait for the lighttable. Press **Cmd+K** (macOS) / **Ctrl+K**
   (Linux) → the palette overlay appears, focused, on top of the main window.
2. Press the shortcut again → the palette hides. (Toggle behaviour.)
3. With the main window editing a canvas, press the shortcut → focus moves to
   the palette but the canvas is not disrupted; Esc/hide returns focus.
4. Tray → **Quick open…** → the same palette overlay opens (shortcut-free path).
5. Set `PENPOT_LOCAL_PALETTE_SHORTCUT=ctrl+shift+p`, relaunch → that combo now
   toggles the palette; the default no longer does.
6. Tray → **Checkpoint now** → native dialog reports the commit (or the loud
   refusal on a dirty repo). Confirm with `git -C <vault> log`.

## Reported numbers
- **Palette ranking latency:** ~5.2 ms to rank the full 1104-item corpus
  (1000 boards + 100 files + 4 projects) for one query, in-process
  (`palette::tests::ranking_is_deterministic_and_fast_at_vault_scale`,
  `PALETTE_RANK_LATENCY items=1104 took_us=5201`). Over HTTP the gate reports
  the end-to-end `tookMs` per query.
- **Grid fix holds scroll at 1000 boards:** gate `(d)` scrolls the 1000-board
  `/__home`, focuses a mid-grid card, waits out a full periodic refresh, and
  asserts `scrollY` unchanged (±2 px) and the same card still focused.
